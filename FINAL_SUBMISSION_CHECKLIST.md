# Final Deliverable Checklist

Use this checklist before submitting your GitHub URL.

## Required Files

- [ ] Repository is public on GitHub.
- [ ] Final notebook is named exactly `main_notebook.ipynb`.
- [ ] `checkpoints/checkpoint_1.ipynb` is included.
- [ ] `checkpoints/checkpoint_2.ipynb` is included.
- [ ] Supporting scripts/modules used by the notebook are committed.
- [ ] `requirements.txt` is in the repository root.
- [ ] `.gitignore` is present.

## README Requirements

- [ ] Project title and one-paragraph overview included.
- [ ] Clear pointer to `main_notebook.ipynb` near top.
- [ ] Research questions listed.
- [ ] Project video link is included and clickable.
- [ ] Dataset source and preprocessing notes are included.
- [ ] Reproduction steps are included (Colab context + run order).
- [ ] Python version from Colab is listed.
- [ ] Key dependency versions are listed.
- [ ] Repo structure tree is included.
- [ ] Short results summary is included.

## Colab Export Steps

- [ ] Run `!pip freeze > requirements.txt` in Colab.
- [ ] Download and place the exported `requirements.txt` at repo root.
- [ ] Run `!python --version` and copy version into README.

## Submission Safety Check

- [ ] Open repo in incognito and verify public visibility.
- [ ] Open `main_notebook.ipynb` from README link and verify it loads.
- [ ] Confirm video link works without login.
- [ ] Submit the repository URL.
